**Шаг не обязательный, может быть пропущен**

* Создать виртуальное окружение: `python3 -m venv venv`
* Активировать виртуальное окружение: 
  * Для Bash/Zsh: `source venv/bin/activate`
  * Для Windows: `venv\Scripts\activate`
